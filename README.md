# 알려진 오류
## Claude Code Action 리포지토리 업데이트로 인해 발생하는 오류

책 **128쪽 「Issue 트리아지용 커스텀 슬래시 명령 만들기」**에서는 Claude Code Action에서 제공하는 label-issue.md 파일을 복사하여 사용합니다.

하지만 Claude Code Action 리포지토리가 업데이트되면서, 최신 label-issue.md 파일을 사용할 경우 추가 설정이 필요하게 되었습니다.

책에 설명된 절차 그대로 따라 하려면, 이 책 집필 당시의 label-issue.md 파일을 사용하세요.

https://github.com/anthropics/claude-code-action/blob/69dec299f882fef0fff1652a1309b7e9771b9f98/.claude/commands/label-issue.md

최신 label-issue.md 파일을 사용하는 경우에는 파일 내용을 확인한 후, 필요한 스크립트를 추가로 배치하는 등 필요한 설정을 진행해야 합니다.
