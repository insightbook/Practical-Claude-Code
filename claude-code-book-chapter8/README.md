# claude-code-book-chapter8

본 저장소는 [「실전 Claude Code 입문 - 현장에서 활용하기 위한 AI 코딩의 사고법」](https://www.amazon.co.jp/dp/4297153548)의 샘플 코드를 관리하는 GitHub 저장소입니다.

저장소에 있는 코드와 프롬프트의 자세한 설명은 책을 참고하기 바랍니다.

## 주의 사항

저장소 내용은 피드백을 통해 개선된 프롬프트로 변경되는 경우가 있습니다.

## 사용법

### 1. 저장소 클론

```bash
git clone [현재 저장소] claude-code-book-chapter8
cd claude-code-book-chapter8
```

### 2. Dev Container를 통해 열기

Visual Studio Code에서 Reopen in Container를 선택하면, 자동으로 다음과 같이 환경 구축이 이루어집니다.

- Node.js LTS 환경 구축
- npm install 실행
- Claude Code 최신판 설치

※ Dev Container를 이용하기 전에 Docker 설치가 필요합니다.
