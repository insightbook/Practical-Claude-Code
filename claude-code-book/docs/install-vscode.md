# Visual Studio Code 설치 방법

Visual Studio Code는 Microsoft에서 제공하는 코드 편집기입니다.

## 다운로드

공식 사이트에서 인스톨러를 다운로드합니다.

<https://code.visualstudio.com/>

사용 중인 OS(Windows / macOS)에 맞는 인스톨러를 선택합니다.

## 설치 절차

### Windows

1. 다운로드한 인스톨러를 실행합니다.
2. 설치 마법사의 안내에 따라 설치를 완료합니다.

### macOS

1. 다운로드한 파일을 열고 Visual Studio Code.app을 애플리케이션 폴더로 복사합니다.
2. 애플리케이션 폴더에서 Visual Studio Code를 실행합니다.

## 명령줄에서 실행 설정하기 (권장)

터미널에서 `code` 명령어로 Visual Studio Code를 실행할 수 있도록 설정합니다. 터미널에서 다음 명령어를 실행하여 버전이 표시되면 사용 가능한 상태입니다.

```bash
code --version
```

`code` 명령어를 사용할 수 없다면 다음 해결 방법을 시도합니다.

- 터미널을 한 번 닫았다가 다시 엽니다.
- Visual Studio Code의 명령어 팔레트(`Ctrl + Shift + P` / `Cmd + Shift + P`)에서 Shell Command: Install 'code' command in PATH를 실행합니다.
